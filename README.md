# Pratham Portfolio 

A modern and responsive developer portfolio showcasing my skills, projects, certifications, and journey as an aspiring Full-Stack Developer.

## Live Website

https://pratham-portfolio-tech.vercel.app

##  Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* Framer Motion
* Vercel

##  Features

* Responsive Design
* Interactive UI
* Smooth Animations
* Project Showcase
* Skills & Certifications
* Contact Section

##  Run Locally

```bash
npm install
npm run dev
```
